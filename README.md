# fireCrudAuth

>  FIREBASE- VUE.JS - VUETIFY : CRUD w/Auth

## Build Setup

``` bash
## Install dependencies
npm install

## Serve on localhost:8080
npm run dev

## Build for production with minification
npm run build

## Build for production and view the bundle analyzer report
npm run build --report
```
